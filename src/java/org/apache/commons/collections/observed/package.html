This package contains collection decorators that allow you to register to listen to changes in the underlying collection. A standard listener based event system is supplied, but alternatives can be added, see ModificationHandler.

Each decorator can be constructed using a static decorate() method on the ObservedXxx class. They can also be constructed from the XxxUtils class (where Xxx is the collection type).

List observedList = ObservedList.decorate(new ArrayList(), listener);
OR
List observedList = ListUtils.observedList(new ArrayList(), listener);
